脚本工具说明
==============

双击对应的 bat 文件即可运行。
所有脚本分为4个入口，覆盖从安装到日常使用的全流程。


================================================================
一、首次安装（新省份只需做一次）
================================================================

  定额安装.bat
  ─────────────────────────────────
  新省份初始化时使用，里面包含两个子功能：

  [1] 导入定额库（必选）
      把省份定额Excel导入数据库，自动识别专业、生成索引。
      对应子脚本：一键导入定额.bat

  [2] 导入定额规则（可选）
      把定额说明文本（txt文件）导入系统，提升匹配准确率。
      文件放在 knowledge\定额说明\ 对应省份文件夹下。
      对应子脚本：导入定额规则.bat

  安装完成后，日常使用不需要再运行此工具。


================================================================
二、日常使用
================================================================

  Jarvis.bat （最常用）
  ─────────────────────────────────
  核心功能：给清单自动匹配定额。

  操作步骤：
    1. 双击运行，选择省份/定额版本
    2. 把清单Excel文件拖进窗口，按回车
    3. 系统自动执行：匹配 → 审核 → 纠正
    4. 完成后可选择打开结果Excel或处理下一个文件

  输出文件在 output\ 目录下。


================================================================
三、学习管理（让系统越来越准）
================================================================

  学习管理.bat
  ─────────────────────────────────
  集中管理所有"让系统学习"的功能，分为两类：

  ● 日常学习（喂数据给系统）

    [1] 导入带定额清单（最常用的学习方式）
        把已经套好定额的工程量清单Excel喂给系统。
        系统会自动学习清单和定额的对应关系，下次遇到相似清单直接命中。
        支持连续导入多个文件，支持切换省份。
        对应子脚本：导入带定额清单.bat

    [2] 导入修正结果
        Jarvis匹配完成后，你在广联达里手工修正了部分定额。
        把修正前（Jarvis输出）和修正后（你保存的）两个Excel都拖进来，
        系统自动对比差异并学习你的修正，下次匹配同类清单更准确。
        对应子脚本：导入修正.bat

    [3] 快捷笔记
        记录零散的造价知识点。比如论坛看到的经验、微信群里的讨论、
        AI对话中学到的技巧等。支持直接输入或批量导入txt文件。
        笔记会在维护方法卡片时被自动整合。
        对应子脚本：快捷笔记.bat

  ● 定期维护（保证数据质量）

    [4] 候选层审核
        系统自动纠正的数据会先存到"候选层"（未经人工确认）。
        这个工具让你逐条审核：确认 / 跳过 / 删除。
        确认后的数据晋升为"权威层"，下次匹配可以直通命中。
        建议：每次Jarvis运行后审核一次。
        对应子脚本：候选层审核.bat

    [5] 经验库体检
        用最新的审核规则回扫经验库，找出历史错误数据。
        发现问题的条目会被降级（不删除），不再参与直通匹配。
        有两种模式：只看报告（不改数据）或 自动修复（降级问题条目）。
        建议：每月运行一次。
        对应子脚本：经验库体检.bat

    [6] 维护方法卡片
        根据积累的经验+规则+笔记，自动生成/更新方法论卡片。
        卡片越丰富，清单匹配定额越准确、越智能。
        有三种模式：预览（不消耗API）、增量更新（推荐）、全量刷新。
        对应子脚本：维护方法卡片.bat


================================================================
四、评测工具（检验匹配质量）
================================================================

  评测工具.bat
  ─────────────────────────────────
  检验系统匹配质量，看系统是越来越准还是在退化。

    [1] 对比分析
        把人工已经套好定额的Excel拖进来，系统会：
        a) 用Jarvis重新匹配这份清单
        b) 逐条对比Jarvis结果和人工结果
        c) 生成差异报告Excel
        d) 可选：把人工结果导入经验库
        对应子脚本：对比分析.bat

    [2] 准确率趋势
        每次运行Jarvis后系统会自动记录关键指标。
        这个工具展示历次运行的准确率变化，帮你判断系统状态。
        对应子脚本：准确率趋势.bat


================================================================
五、开发者工具（scripts/dev/ 目录）
================================================================

  日常不需要使用，仅供 Claude Code 开发调试。

  system_health.bat     系统体检+代码审查（主目录下，有菜单可选模式）
  release_check.bat     发布前一键体检（体检+基准对比+Jarvis冒烟）
  dev/系统体检.bat      快捷方式，转发到 system_health.bat
  dev/代码审查.bat      快捷方式，转发到 system_health.bat review
  dev/发布体检.bat      快捷方式，转发到 release_check.bat
  dev/ai_pair_guard.bat Claude修复 + Codex检查流水线
  dev/send_to_claude_fix.bat  把修复任务发给Claude


================================================================
推荐使用流程
================================================================

  首次使用新省份：
    定额安装.bat → 导入定额库 → 导入定额规则（可选）

  日常工作：
    Jarvis.bat → 拖入清单 → 等待匹配 → 打开结果检查

  积累经验：
    学习管理.bat → [1]导入带定额清单 或 [2]导入修正结果

  定期维护（建议每月）：
    学习管理.bat → [4]候选层审核 → [5]经验库体检 → [6]维护方法卡片

  质量检查：
    评测工具.bat → 对比分析 或 准确率趋势
    发布前执行：release_check.bat ci search all --no-pause


================================================================
子脚本说明
================================================================

  所有子脚本仍保留在 scripts/ 目录下，可以独立双击运行。
  导航页（学习管理/定额安装/评测工具）只是调用它们，不会重复逻辑。

  如果你已经熟悉某个子功能，直接双击对应的子脚本也完全可以，
  不一定要通过导航页进入。
