## 专业: C10

### 填料套管安装

- 基于 12 条样本，确认率 100%
- 关键词: 填料套管, 钢制套管, 普通套管

选定额方法：
1. 一般填料套管/普通钢制套管 → "一般填料套管制作安装"定额（C10-4-63至C10-4-75系列）
2. 关键取档参数：公称直径（mm），向上取档
3. 两种规格表达方式：
   - DN格式（如DN65）→ 直接用DN值
   - 宽×高格式（如1700×500）→ 取较小的数值作为公称直径
4. 档位对应：
   - DN≤50 → C10-4-63
   - DN≤80 → C10-4-65
   - DN≤100 → C10-4-66
   - DN≤150 → C10-4-68
   - DN≤300 → C10-4-70
   - DN≤450 → C10-4-73
   - DN≤500 → C10-4-74
5. 不需要关联定额

**常见错误:**
- 填料套管≠防水套管（防水套管有专门的定额）
- 宽×高格式中，较小值是管径，较大值是套管长度
- 普通钢制套管和一般填料套管用同一套定额

### 截止阀/闸阀安装

- 基于 14 条样本，确认率 100%
- 关键词: 截止阀, 闸阀, 金属阀门, 丝扣连接, 法兰阀门, Y型过滤器

选定额方法：
1. 截止阀、闸阀、止回阀、Y型过滤器等常规阀门 → "焊接法兰阀门"定额（C10-5-33至C10-5-45系列）
2. 关键取档参数：公称直径DN，向上取档
3. 档位对应：
   - DN≤32 → C10-5-33
   - DN≤40 → C10-5-34
   - DN≤50 → C10-5-35
   - DN≤65 → C10-5-36
   - DN≤80 → C10-5-37
   - DN≤100 → C10-5-38
   - DN≤150 → C10-5-40
   - DN≤200 → C10-5-41
   - DN≤250 → C10-5-42
4. 注意：丝扣连接的小口径阀门（DN15-32）也用法兰阀门定额
5. 不需要关联定额

**常见错误:**
- 材质（铜芯/碳钢）和连接方式（丝扣/沟槽/法兰）不影响定额选择，统一用"焊接法兰阀门"
- DN15/20/25都在DN32以内（C10-5-33）
- Y型过滤器也参照阀门定额

### 蝶阀安装

- 基于 6 条样本，确认率 100%
- 关键词: 蝶阀, 对夹式

选定额方法：
1. 蝶阀 → "焊接对夹式法兰阀门"定额（C10-5-51至C10-5-57系列）
2. 关键取档参数：公称直径DN，直接按DN值取档
3. 档位对应（每档对应一个DN值）：
   - DN≤100 → C10-5-51
   - DN≤125 → C10-5-52
   - DN≤150 → C10-5-53
   - DN≤200 → C10-5-54
   - DN≤250 → C10-5-55
4. 不需要关联定额

**常见错误:**
- 蝶阀用"焊接对夹式法兰阀门"定额，不要选成普通"焊接法兰阀门"
- DN值直接对应，不需要计算周长

## 专业: C4

### 电力电缆头安装

- 基于 4 条样本，确认率 100%
- 关键词: 电缆头, 电力电缆头, 终端头, WDZN, YJY

选定额方法：
1. 电力电缆头 → "户内干包式非铠装电力电缆终端头制作安装"定额（C4-8-230至C4-8-240系列）
2. 关键取档参数：芯数×截面积（从电缆型号中提取）
3. 先确定电压等级：1kV以下（WDZN-YJY为低压电缆）
4. 档位按芯数×截面取：
   - 4×6mm2 → C4-8-232
   - 4×16mm2 → C4-8-235
   - 5×16mm2 → C4-8-236
5. 截面取档规则：向上取档（如5×10实际用5×16的定额）
6. 不需要关联定额

**常见错误:**
- 电缆型号中提取芯数和截面（如WDZN-YJY-5x16 → 5芯16mm2）
- 截面向上取档（5×10选5×16）
- 铠装和非铠装电缆头定额不同，注意区分

### 配电箱安装

- 基于 5 条样本，确认率 100%
- 关键词: 配电箱, AL, 箱体安装

选定额方法：
1. 配电箱 → "配电箱箱体安装"定额（C4-4-35至C4-4-41系列）
2. 关键取档参数：半周长 = (宽+高) / 1000，单位m
3. 安装方式：明装/暗装（影响定额选择）
4. 档位对应（明装）：
   - 半周长≤1.5m → C4-4-35
   - 半周长≤2.0m → C4-4-36
   - 半周长≤2.5m → C4-4-37
   - 半周长≤3.0m → C4-4-38
5. 计算示例：600×500×300的配电箱，半周长=(600+500)/1000=1.1m，选2.5m以内
6. 不需要关联定额，但需注意箱内元器件另计

**常见错误:**
- 半周长是(宽+高)/1000转米，不是周长也不是面积
- 明装和暗装的定额编号不同
- 配电箱编号如AL1+1中的数字是回路编号，不影响定额选择

### 钢制桥架安装

- 基于 17 条样本，确认率 100%
- 关键词: 桥架, 钢制桥架, 槽式桥架

选定额方法：
1. 确定桥架材质：钢制桥架 → "钢制槽式桥架"定额（C4-11-248至C4-11-252系列）
2. 关键取档参数：宽+高之和，单位mm
3. 档位对应：
   - 宽+高≤150mm → C4-11-248
   - 宽+高≤400mm → C4-11-249
   - 宽+高≤600mm → C4-11-250
   - 宽+高≤800mm → C4-11-251
   - 宽+高≤1000mm → C4-11-252
4. 计算示例：200×100的桥架，宽+高=300mm，选400以下(C4-11-249)
5. 超高桥架与普通桥架用同一套定额
6. 不需要关联定额

**常见错误:**
- 宽+高是直接相加（不是周长、不是面积）
- 钢制桥架≠铝合金桥架，材质不同定额不同
- "超高"标注不影响定额选择

## 专业: C6

### 压力表安装

- 基于 6 条样本，确认率 100%
- 关键词: 压力表

选定额方法：
1. 压力表 → "压力表 就地"定额，编号固定为 C6-1-49
2. 不需要按DN取档（所有规格都用同一条定额）
3. DN值指的是安装管道的口径，不影响压力表定额选择
4. 不需要关联定额

**常见错误:**
- 压力表不需要按DN分档，统一用C6-1-49
- 不要和温度计（C6-1-2）混淆

## 专业: C7

### 多叶调节阀安装

- 基于 5 条样本，确认率 100%
- 关键词: 多叶调节阀, 对开多叶, 调节阀

选定额方法：
1. 电动对开多叶调节阀 → "多叶调节阀安装"定额（C7-6-16至C7-6-19系列）
2. 关键取档参数：周长 = (宽+高) × 2，单位mm
3. 档位对应：
   - 周长≤1200mm → C7-6-16
   - 周长≤2400mm → C7-6-17
   - 周长≤3600mm → C7-6-18
4. 计算示例：400×250的阀，周长=1300mm，选2400以内(C7-6-17)
5. 不需要关联定额

**常见错误:**
- 多叶调节阀和防火调节阀是不同定额系列（C7-6-16 vs C7-6-1），不要混淆
- 电动/手动不影响定额选择

### 百叶风口安装

- 基于 24 条样本，确认率 100%
- 关键词: 百叶风口, 单层百叶, 双层百叶, 防雨百叶, 风口

选定额方法：
1. 确定风口类型：单层百叶/双层百叶/防雨百叶/防雨防虫百叶，统一对应"百叶风口安装"定额（C7-7-2至C7-7-6系列）
2. 关键取档参数：周长 = (宽+高) × 2，单位mm
3. 档位对应：
   - 周长≤1200mm → C7-7-2
   - 周长≤1800mm → C7-7-3
   - 周长≤2400mm → C7-7-4
   - 周长≤3200mm → C7-7-5
   - 周长≤6000mm → C7-7-6
4. 计算示例：250×250的风口，周长=1000mm，选1200以内(C7-7-2)；500×500的风口，周长=2000mm，选2400以内(C7-7-4)
5. 不需要关联定额

**常见错误:**
- 单层/双层/防雨百叶都用同一套定额，不要因为类型不同选不同系列
- 周长计算要注意是(宽+高)×2，不是(宽×高)面积
- 规格给的是宽×高（mm），需要自己算周长再取档

### 防火调节阀安装

- 基于 11 条样本，确认率 100%
- 关键词: 防火阀, 防烟防火, 防火调节阀, 常闭电动

选定额方法：
1. 70℃防火阀和280℃常闭电动防烟防火调节阀统一用"防火调节阀安装"定额（C7-6-1至C7-6-5系列）
2. 关键取档参数：方/矩形周长 = (宽+高) × 2，单位mm
3. 档位对应：
   - 周长≤1200mm → C7-6-1
   - 周长≤1800mm → C7-6-2
   - 周长≤2400mm → C7-6-3
   - 周长≤3200mm → C7-6-4
   - 周长≤4400mm → C7-6-5
4. 计算示例：500×320的阀，周长=1640mm，选1800以内(C7-6-2)
5. 不需要关联定额

**常见错误:**
- 70℃和280℃防火阀用同一套定额，不要区分温度选不同系列
- 电动/手动不影响安装定额选择
- 取档参数是周长，不是面积

## 专业: C9

### 消防钢管安装

- 基于 6 条样本，确认率 100%
- 关键词: 镀锌钢管, 沟槽连接, 消防管

选定额方法：
1. 消防用镀锌钢管（沟槽连接）→ "钢管(沟槽连接)"定额（C9-1-11至C9-1-19系列）
2. 关键取档参数：公称直径，直接对应
3. 档位对应：
   - DN65 → C9-1-13
   - DN100 → C9-1-15
   - DN150 → C9-1-17
4. 注意：消防管道用C9（消防册），不用C10（给排水册）
5. 超高标注不影响定额选择
6. 不需要关联定额（管卡/试压另外算）

**常见错误:**
- 消防管道用C9册，给排水管道用C10册，不要搞混
- "超高"标注不影响定额编号选择
- 沟槽连接和丝扣连接的定额不同
