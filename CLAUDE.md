# 项目说明（模板）

## 用户背景

我是非程序员，通过 Claude Code 辅助编程。请注意：

- 用通俗易懂的方式解释技术概念
- 不要假设我了解编程术语，遇到专业词汇请附上解释
- 改动代码前先说明你打算做什么，等我确认再动手
- 如果有多种实现方案，列出各自优缺点让我选择

## 项目结构约定

- 源代码放在项目根目录或 `src/` 下
- 数据文件放在 `data/` 下
- 输出结果放在 `output/` 下
- 配置文件放在项目根目录

## 工作流程

1. 我说需求，你先确认理解是否正确
2. 你提出方案，简单解释思路
3. 我确认后你再写代码
4. 写完后告诉我怎么运行、预期结果是什么
