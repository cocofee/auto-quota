# 北京安装定额复核提示

适用场景：

- 配管 SC20 暗敷
- 电缆沿桥架敷设

复核原则：

1. 先识别安装路径，再判断主材类型。
2. 不能把桥架敷设误判成导管敷设。
3. 单位和工作内容冲突时，优先人工复核。
