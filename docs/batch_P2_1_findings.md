# 批次 P2-1 Findings（灯具硬映射偏移风险）

## 问题清单

### [P2] 直管灯/灯管被错误映射到 LED灯带

- **位置**: `src/query_builder.py:179-181`
- **严重程度**: P2（搜索召回偏移）
- **原因**:
  原正则 `r'直管|灯管|线槽灯'` 把三种不同灯具统一映射为 `"LED灯带 灯管式"`。

  - **直管灯 / 灯管**：管状灯具（不论 LED 还是荧光），安装工艺走荧光灯安装定额。
    直管灯管需要安装灯座、灯架，与荧光灯安装完全一致。
  - **线槽灯**：安装在线槽内的线性 LED 灯具，安装工艺接近 LED 灯带。

  两者安装工艺不同，对应不同定额家族，不应合并映射。

- **影响范围**: 所有包含"直管灯"或"灯管"的清单项（安装专业 C4 册）
- **复现**: `_normalize_bill_name("直管灯")` 返回 `"LED灯带 灯管式"` → BM25 搜索偏移到灯带定额

### [P2] 长尾灯具缺乏回归测试覆盖

- **位置**: `tests/test_query_builder_lamp_rules.py`
- **严重程度**: P2（质量保障缺口）
- **原因**:
  原测试只覆盖了常见灯具（吸顶灯、荧光灯、应急灯、标志灯），
  长尾灯具（筒灯、地脚灯、庭院灯、防爆灯、井道灯等）没有回归测试。
  后续修改灯具规则时无法自动检测到长尾灯具的偏移。
