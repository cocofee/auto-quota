# Jarvis 自动套定额系统 — 使用说明

## 系统是什么

读取工程量清单Excel，自动匹配最合适的定额子目，输出广联达可导入的Excel。
系统会从你的每一次使用中学习，用得越多越准确。

---

## 首次安装

### 1. 安装定额库

双击 `scripts\定额安装.bat`，按提示：
1. 选择省份（如"北京2024"）
2. 选[1]导入定额库 → 把定额Excel拖进去
3. 选[2]导入定额规则（可选，提升准确率）

### 2. 喂历史数据

双击 `scripts\学习管理.bat`，选[1]导入带定额清单：
- 把你手里**已经套好定额的**Excel拖进去
- 系统会学习"这条清单对应什么定额"
- 导入越多，系统越准确

## 日常使用（核心流程）

### 完整闭环：套定额 → 广联达审核 → 系统学习

```
第1步：Jarvis套定额
    ↓
第2步：导入广联达，重点看[需复核]的条目，改好导出
    ↓
第3步：学习回收（把修正结果喂回系统）
    ↓
下次匹配同样的清单 → 更准更快
```

### 第1步：Jarvis套定额

双击 `scripts\Jarvis.bat`：
1. 选择省份
2. 把清单Excel拖进窗口，回车
3. 等待自动完成（匹配→审核→纠正→存经验库 全自动）
4. 在 `output\` 目录找到结果Excel

**结果Excel里的标记说明：**

| J列（推荐度） | 含义 | 你需要做什么 |
|--------------|------|------------|
| ★★★推荐(95%) | 高置信度，基本可信 | 一般不用管 |
| ★★参考(72%) | 中等置信度 | 建议看一眼 |
| ★待审(45%) | 低置信度 | 必须检查 |

| K列（匹配说明） | 含义 |
|----------------|------|
| [需复核] xxx | 系统不太确定，请重点看这条 |
| （无前缀） | 系统比较有把握 |

**你只需要重点看带[需复核]标记的条目**，大约占总量的3-5%。

### 第2步：导入广联达审核

1. 把结果Excel导入广联达
2. 在广联达里筛选/查看带[需复核]的条目
3. 发现不对的，在广联达里改正
4. 改完导出Excel

### 第3步：学习回收

把**Jarvis原始输出**和**你改完的版本**拖到 `tools\学习回收.bat` 上：
- 第1个文件：Jarvis输出的原始Excel（在output目录里）
- 第2个文件：你在广联达改完后导出的Excel

系统会自动对比：
- 你没改的 → 系统记住"这个匹配是对的"
- 你改了的 → 系统学习"下次应该用你改的定额"

**下次遇到同样的清单，系统就不会再犯同样的错了。**

## 系统如何自动变聪明

### 两层数据机制

| 层 | 来源 | 作用 |
|----|------|------|
| 权威层 | 你确认/纠正的数据 | 直接用于匹配（秒出结果） |
| 候选层 | 系统自动匹配的数据 | 仅做参考，不直接用 |

### 自动晋升规则

候选层的数据如果在多个项目中**反复匹配到同样的结果**，系统会自动信任它：

| 置信度 | 需要多少次一致命中 | 才自动晋升为权威层 |
|--------|------------------|-----------------|
| 95分以上 | 2次 | 很快就能用 |
| 90-94分 | 3次 | 几个项目后 |
| 85-89分 | 5次 | 多个项目验证 |
| 85分以下 | 不自动晋升 | 必须你亲自确认 |

**你不需要手动管这些**，系统全自动处理。

## 工具一览

### 主要工具（scripts目录）

| 工具 | 用途 | 使用频率 |
|------|------|---------|
| `Jarvis.bat` | 套定额（核心功能） | 每个项目都用 |
| `学习管理.bat` | 导入数据、维护经验库 | 有新数据时用 |

| 功能 | 说明 |
|------|------|
| 导入带定额清单 | 把已套好定额的Excel喂给系统 |
| 导入修正结果 | 对比原始和修正版，学习差异 |
| 候选层审核 | 审核系统自动匹配的数据 |
| 经验库体检 | 用最新规则回扫历史数据 |

| 工具 | 说明 |
|------|------|
| `学习回收.bat` | 拖拽两个文件，自动对比学习 |

| `system_health_check.py` | 系统体检（检查是否正常） |

## 常见问题

### Q: 导入广联达后格式乱了？
A: 结果Excel严格遵循广联达标准格式（宋体9号、序号+编码+名称结构），正常情况可直接导入。如果格式有问题，检查是否是清单Excel本身格式特殊。

### Q: 系统匹配准确率多少？
A: 刚开始用大概85%左右。随着你不断使用+学习回收，准确率会持续提升。经验库积累够多后，常见清单基本能做到95%+直通命中。

### Q: 换了省份怎么办？
A: 运行 `定额安装.bat` 导入新省份的定额。经验库的学习成果跨省份共享（通用知识库），但具体定额编号需要重新积累。

### Q: 经验库数据有问题怎么办？
A: 在 `学习管理.bat` 里选[5]经验库体检，系统会自动扫描并降级有问题的数据。
